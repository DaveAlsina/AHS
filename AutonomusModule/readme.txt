Esta rama del proyecto sigue en investigación: 

Lo que busca realizar es crear módulos autónomos(energéticamente) que tomen mediciones, 
se conecten a wifi y las envíen. 

Para realizar esta tarea se requiere: 
	1)	Progresar el desarrollo de un módulo que funcione con energía solar.
	2)	Aumentar el ahorro energético, para aumentar la duración de las baterías 
		y con ello la autonomía de los modulos de mediciones.
